What is HTML?

HTML (HyperText Markup Language) is the standard language used to create the structure of web pages.

What is CSS used for in web development?

CSS (Cascading Style Sheets) is used to style and layout web pages, including design, colors, fonts, and positioning.

What is the role of JavaScript in web development?

JavaScript adds interactivity and dynamic behavior to web pages, such as animations, form validation, and API handling.
